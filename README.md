# Certainti Backend APIs

Folder structure
C:.
└───certainti-apis
    └───v1
        ├───data
        ├───public
        │   └───temp
        └───src
            ├───controllers
            ├───middlewares
            ├───models
            ├───queries
            ├───routes
            ├───setups
            │   ├───db
            │   ├───logging
            │   └───server
            └───utils

## Getting Started

Install dependencies and start the server:

```bash
npm install
npm start
```

For development with automatic reloads use `npm run dev`.
